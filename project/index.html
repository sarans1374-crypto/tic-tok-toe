Responsive Landing Page with Interactive Navigation

This responsive layout was developed for SkillCraft Task 01 to demonstrate front-end best practices.

Task Overview

This project was created as part of SkillCraft Task 01, focusing on building a responsive landing page with interactive navigation using HTML, CSS, and JavaScript.

Design Approach

The design uses light neutral tones, consistent padding, and clear typography to maintain a clean and professional layout that aligns with real-world front-end standards.

Responsive Layout

Built using CSS Flexbox and media queries, this page adapts smoothly across devices, ensuring readability and usability on phones, tablets, and desktops.

Interactive Features

The navigation bar remains fixed at the top and changes color on scroll. Navigation links respond on hover, enhancing the overall user experience.
